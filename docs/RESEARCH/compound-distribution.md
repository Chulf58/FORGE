# Research: Compound Engineering Distribution

**Compound Engineering** is a methodology and Claude Code plugin by Every.to that structures AI-assisted development into a Plan → Work → Review → Compound loop. It is distributed as:

1. **Claude Code Plugin** — Official marketplace entry (`EveryInc/compound-engineering-plugin`)
2. **npm package** — `@every-env/compound-plugin` for CLI-based installation
3. **Multi-tool conversion** — Bun/TypeScript CLI converts the plugin for Codex, Windsurf, Gemini CLI, and others

Users install via `/plugin marketplace add` or `bunx @every-env/compound-plugin install`.

The plugin includes 26 agents, 23 commands, and 13 skills. Core principle: each feature should make the next easier to build.
